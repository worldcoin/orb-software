# Pinned NXP Root and Intermediate Certificates

These .crt files come from NXP's website, and are linked to by [AN12436][AN12436],
section 3.11.3.

* `63709315050002.crt`: the [root][root] nxp cert.
  (Subject OU=Plug and Trust, O=NXP, CN=NXP RootCAvE506)
* `63709315060003.crt`: the [intermediate][intermediate] nxp cert.
  (Subject OU=Plug and Trust, O=NXP, CN=NXP Intermediate-AttestationCAvE206)
