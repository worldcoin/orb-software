# Example se050 data

These files come from a real se050 on a (dev) orb, with one exception:

`2A66F1B2_bad_signature.crt` is a file generated by claude. Claude took the
original crt file, and replaced its signature field with a signature signed by
a different issuer private key (i.e. not the one that matches the intermediate
CA). This is to help unit tests confirm that the code rejects
attacker-controlled keys.
